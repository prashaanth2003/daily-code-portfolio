# HackerRank: Solve Me First

- **Difficulty**: Easy
- **URL**: [HackerRank: Solve Me First](https://www.hackerrank.com/challenges/solve-me-first/problem)

## Problem Statement

Complete the function `solveMeFirst` to compute the sum of two integers.

### Input Format
Two integers, `a` and `b`.

### Output Format
An integer representing the sum of `a` and `b`.

### Example
```
a = 2
b = 3
Output: 5
```

## Solution Approach
The simplest introduction to programming constraints. We take two parameters `a` and `b` and return their sum using the standard addition operator `+`.

## Complexity Analysis
- **Time Complexity**: $O(1)$
- **Space Complexity**: $O(1)$
